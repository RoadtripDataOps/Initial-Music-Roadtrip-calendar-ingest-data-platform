# Source Snapshot Notes

Generated: 2026-06-04

This file contains short factual notes from official/reference sources. It intentionally does not reproduce full vendor documentation.

## Ticketmaster

Official developer portal says Ticketmaster offers event discovery and commerce APIs; Discovery and Commerce are available after registration/API key, while Partner API is restricted to official distribution relationships. Discovery API is global and sources multiple platforms including Ticketmaster, TicketWeb, Universe, FrontGate, and Ticketmaster Resale.

## SeatGeek

Platform overview describes a live events dataset including venue lat/lon. API terms define Developer Portal, Documentation, SeatGeek APIs/materials, and restrict unauthorized/systematic downloading or scraping outside permitted usage.

## DICE

Official Ticket Holders API is GraphQL and focused on partner ticket-holder, access, and finance/BI workflows for partner events.

## viagogo / StubHub

Both have public catalog API docs with OAuth2 and endpoints for categories, events, and venues; catalog sync supports updated_since/resource_version-style workflows.

## Seated

Seated API docs list base URL https://api.cm.com/seatedapi and authenticate via X-CM-PRODUCTTOKEN. Focus is ticketing data insights.

## Bandsintown

Bandsintown API docs expose artist info and artist events; artist events include date/time, venue name/location, ticket links, lineup, description, title, and Bandsintown event page.

## Eventim / See Tickets US

Affiliate Network article describes an API-based feed that allows marketplaces to ingest event information.

## TicketWeb

TicketWeb pages say WordPress plugin and direct API access can sync event details, but exact endpoint docs are not public.

## AXS / Etix / SuiteHop / Sofar

No complete public event API reference was found. Treat these as partner-only or link-only until docs/credentials are supplied.
